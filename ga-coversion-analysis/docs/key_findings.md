KEY FINDINGS — January 2017 GMS Data

Sample size: 61,952 sessions (43,615 desktop / 18,337 mobile)
Converting sessions: 688 (614 desktop / 74 mobile)

CONVERSION RATE
- Desktop: 1.41% (95% CI: 1.30% - 1.52%)
- Mobile: 0.40% (95% CI: 0.31% - 0.50%)
- Chi-square test p-value: 1.32 × 10^-27
- Relative lift: 3.49x

AVERAGE ORDER VALUE (converting sessions only)
- Desktop: mean $148.54, median $51.00 (n=614)
- Mobile: mean $83.84, median $22.44 (n=73)
- Welch's t-test p-value: 0.019
- Mean difference: $64.70 (77% relative)

COMBINED IMPACT
- Desktop revenue per session: 1.41% × $148.54 = ~$2.09
- Mobile revenue per session: 0.40% × $83.84 = ~$0.34
- Desktop generates ~6x more revenue per session than mobile

CAVEATS
- Observational data, not a randomized A/B test (device is self-selected)
- Revenue is heavily right-skewed; means inflated by outliers
- Mobile sample (n=73 converters) limits statistical power for AOV comparison
- Mann-Whitney U test would be more appropriate for AOV given non-normality
